## 4. 选型建议

**选型标准**（按优先级）：
1. 严格 recall 最高（CWE 也对，反映真实技能提升）
2. FPR 不恶化（FP 不增加）
3. dev_loss 最低（训练侧泛化最好）
4. CoT-JSON 不一致为 0

**自动推荐**：`lr=1e-5 (baseline)`（综合评分最高）

**下一步**：
- 若推荐配置的严格 recall 显著高于 baseline（>3pp），直接进入 Phase 3（DPO）
- 若提升有限（<2pp），进入 Phase 2：r=32 + rsLoRA + e=2 + lr=1e-4
- 若 FPR 恶化，回到 baseline lr=1e-5 + rsLoRA 作为 Phase 2 起点
